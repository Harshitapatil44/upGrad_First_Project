Python Programming Project :- 

This repository contains Python programs developed as part of the **upGrad Python Programming course**. The project focuses on strengthening programming fundamentals through practical exercises and mini-projects.

Project Overview :- 
The objective of this project is to build a strong foundation in Python by implementing core programming concepts and solving logical problems using real-world inspired tasks.

Concepts Covered :- 
- Variables and Data Types
- Conditional Statements
- Loops (for, while)
- Functions
- Lists, Tuples, Dictionaries, and Sets
- File Handling
- Error and Exception Handling
- Basic Data Analysis Concepts

Features :- 
- Multiple Python programs demonstrating core concepts
- Mini problem-solving exercises
- Practice tasks focused on logic building
- File reading and writing operations
- Error handling using try-except blocks

Technologies Used :- 
- Python
- Standard Python Libraries

Learning Outcomes :- 
- Improved problem-solving and logical thinking
- Better understanding of Python programming fundamentals
- Hands-on practice with debugging and code optimization

How to Run the Project :- 
1. Install Python (version 3.x recommended)
2. Clone the repository ( https://github.com/Harshitapatil44/upGrad_First_Project )
3. Navigate to the project folder
4. Run any Python file
